# Release v1.2.1

## 概要

Google ドライブのファイルブラウザ機能を追加し、フォルダ内のファイルをドラッグ＆ドロップでスロットに配置できるようになりました。

## 変更内容

### 新機能

- **Google ドライブ ファイルブラウザ** — Google ドライブ内のフォルダを階層的に閲覧し、ファイルをドラッグ＆ドロップで起承転結スロットに直接配置できるようになりました。
  - マイドライブのルートから閲覧開始
  - フォルダクリックでサブフォルダへ移動
  - パンくずナビゲーションで上位フォルダへ戻る
  - Google ドキュメント・スプレッドシートをドラッグしてスロットへドロップ
  - モーダルを開いたまま複数ファイルを連続配置可能

### バグ修正

- **`window.prompt()` エラーの修正** (v1.2.0) — Electron で未対応の `window.prompt()` をカスタムモーダルに置換。

### ドキュメント

- **Google OAuth セットアップガイドの追加** (`docs/GOOGLE_OAUTH_SETUP.md`) — 403 `access_denied` エラーの解決方法、テストユーザーの登録手順。
- **README にテストユーザー登録手順を追加** — 手順 5 として追記。

### ビルド・インフラ

- **Linux ビルドをリリースワークフローに追加** — GitHub Actions で macOS / Windows / Linux の全プラットフォームビルドを自動化。
